# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## v1.1.0 (2026-03-01)

### Added
- Batch experiment mode with systematic comparison across multiple SRAC ratios
- Comprehensive chart output for batch experiments
- Fitness quartile analysis (top 25% vs. bottom 25%)
- Result export to pickle files

### Fixed
- Removed dead code, unused imports, and redundant comments
- Corrected LICENSE copyright holders

## v1.0.0 (2026-03-01)

Initial public release — faithful Python 3 port from original Java simulator.

### Added
- N×N grid agent simulation with 16 deterministic memory-1 strategies
- Cellular Automata and Small-World network topologies
- Self-Reputation Awareness Component (SRAC) mechanism
- IPD game engine with cycle detection optimization
- Tkinter GUI with real-time visualization and generation scrubbing
- CLI mode for scripted execution
- Comprehensive README and CONTRIBUTING documentation
